# 📊 Анализ инвестиций в стартапы

## Обзор проекта

В рамках проекта проведён комплексный разведочный анализ данных (EDA) о венчурных инвестициях в стартапы. Цель работы — выявить закономерности финансирования, оценить перспективные рыночные сегменты и типы инвестиций, а также сформулировать рекомендации для инвестиционной компании.

## 📁 Описание наборов данных

В работе использованы два датасета:

### 1. **Данные о стартапах** (`cb_investments.csv`)
Содержит информацию о 54 294 компаниях и 40 признаках:

| Признак | Описание |
|---------|----------|
| `name` | Название компании |
| `market` | Основной рынок / отрасль |
| `funding_total_usd` | Общий объём привлечённых инвестиций в долларах США |
| `status` | Текущий статус компании (operating, closed и др.) |
| `founded_at` | Дата основания |
| `first_funding_at` | Дата первого раунда финансирования |
| `last_funding_at` | Дата последнего раунда финансирования |
| `funding_rounds` | Общее число раундов финансирования |
| `participants` | Количество участников в раундах |
| Различные типы финансирования | Seed, venture, debt financing, angel, grant и др. |

### 2. **Данные о возвратах** (`cb_returns.csv`)
Содержит годовые суммы возвратов по типам финансирования (2000–2014 гг.) в миллионах долларов.

## 🔍 Основные этапы анализа

### 1. Предобработка данных
- Приведение названий столбцов к единому формату
- Преобразование типов данных (даты → datetime, суммы → числовые)
- Обработка пропусков (заполнение «unknown» для категорий, медианное заполнение для числовых)
- Удаление дубликатов
- Фильтрация выбросов и аномалий

### 2. Инжиниринг признаков
- **Классификация по срокам финансирования**:
  - Единичный раунд
  - Срок финансирования ≤ 1 года
  - Срок финансирования > 1 года

- **Сегментация рынков**:
  - **Массовый сегмент**: > 120 компаний в отрасли
  - **Средний сегмент**: 35–120 компаний
  - **Нишевый сегмент**: < 35 компаний

### 3. Анализ выбросов
Выявлены 5 сегментов с наибольшей долей аномальных значений:
1. Недвижимость (Real Estate)
2. Облачные вычисления (Cloud Computing)
3. Развлечения (Entertainment)
4. SaaS
5. Технологии (Technology)

### 4. Анализ типов финансирования
**Наибольшие объёмы привлечены через:**
- Венчурные инвестиции
- Прямые частные инвестиции (private equity)
- Долговое финансирование
- Пост-IPO акционерный капитал

**Наиболее популярные типы (по числу раундов):**
1. Венчурные инвестиции
2. Посевные инвестиции (seed)
3. Долговое финансирование
4. Инвестиции бизнес-ангелов

### 5. Динамика во времени
- **Объём финансирования по годам**: пик приходится на 2012–2013 гг.
- **Средний размер раунда**: историческая динамика
- **Помесячная динамика за 2014 год**: заметно снижение с сентября

### 6. Динамика массовых сегментов (2013–2014)
**Сегменты с наибольшим ростом:**
1. Software (наибольший абсолютный объём)
2. Medical (самый быстрый рост)
3. Technology
4. SaaS
5. Apps
6. Internet

### 7. Анализ возвратов
**Средняя доля возврата по типам инвестиций:**
- Прямые частные инвестиции (private equity): **86 %**
- Долговое финансирование: **81 %**
- Инвестиции ангелов: **64 %**
- Посевные инвестиции: **52 %**
- Венчурные инвестиции: **42 %**

## 💡 Ключевые выводы и рекомендации

### Рекомендуемые отрасли для инвестиций:
- ✅ Software
- ✅ Medical (здравоохранение)
- ✅ Technology
- ✅ SaaS
- ✅ Мобильные приложения
- ✅ Интернет / электронная коммерция

### Рекомендуемые типы финансирования:
1. **Венчурные инвестиции** — наибольшие объёмы, стабильный рост
2. **Долговое финансирование** — устойчивый возврат (в среднем > 80 %)
3. **Прямые частные инвестиции** — самые высокие показатели возврата (~86 %)

### Риски:
- Private equity отличается высокой волатильностью
- Посевные и ангельские инвестиции имеют длительный срок окупаемости
- В некоторых сегментах наблюдаются аномальные суммы (требуется дополнительный due diligence)
