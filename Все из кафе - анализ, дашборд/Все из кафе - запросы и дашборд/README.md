# Проект: Анализ пользовательских метрик сервиса доставки (SQL + DataLens)

Данный проект демонстрирует навыки написания сложных SQL-запросов для расчёта ключевых продуктовых метрик на основе данных о заказах в сервисе доставки еды. В рамках работы были проанализированы данные по городу **Саранск** за период **май – июнь 2021 года**. Итоговые результаты визуализированы в интерактивном дашборде **DataLens**.

---

## 🧠 Задачи и использованные метрики

В ноутбуке последовательно решаются 7 аналитических задач:

| № | Метрика | Описание |
|---|---------|----------|
| 1 | **DAU** (Daily Active Users) | Количество уникальных пользователей, совершивших заказ в день. |
| 2 | **Conversion Rate (CR)** | Доля пользователей, совершивших заказ, от всех активных пользователей в день. |
| 3 | **Средний чек** | Средняя сумма комиссии (revenue * commission) на один заказ по месяцам. |
| 4 | **LTV ресторанов** | Суммарная комиссия, принесённая каждым рестораном за период (LTV). Топ‑3 ресторана. |
| 5 | **LTV по блюдам** | Для двух ресторанов с наибольшим LTV – самые популярные блюда по величине комиссии. |
| 6 | **Retention Rate** | Доля пользователей, вернувшихся в приложение на N‑й день после первого посещения (первые 7 дней). |
| 7 | **Retention по месяцам** | Сравнение кривых удержания для пользователей, пришедших в мае и июне. |

---

## 🛠 Технологии и инструменты

- **Язык запросов:** SQL (синтаксис PostgreSQL)
- **Среда разработки:** Jupyter Notebook (оформление и документирование)
- **Визуализация:** DataLens (дашборд)
- **Ключевые конструкции SQL:**
  - CTE (WITH)
  - JOIN (INNER, USING)
  - Агрегатные функции (COUNT, SUM, ROUND)
  - Оконные функции (OVER, PARTITION BY)
  - Фильтрация (WHERE, BETWEEN)
  - Приведение типов (::numeric, ::date)

---

## 🔍 Основные результаты

- **DAU** – ежедневная динамика числа покупателей.
- **CR** – доля заказчиков среди всех посетителей (колеблется в пределах 0.2–0.4).
- **Средний чек** – рассчитан по месяцам (май и июнь) в разрезе комиссии сервиса.
- **LTV ресторанов** – лидером является сеть с наибольшей суммарной комиссией (данные в запросе №4).
- **Топ блюд** – для двух лидирующих ресторанов выявлены блюда, приносящие максимальный доход.
- **Retention** – построена кривая удержания для первых 7 дней, а также проведено сравнение когорт мая и июня (во второй месяц удержание оказалось выше).

Все расчёты выполнены с использованием единого подхода и могут быть легко адаптированы под другие города или периоды.
